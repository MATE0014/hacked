# InsightsFlow - AI Data Analyzer

A full-stack web application for **zero-config automated data analysis** with AI-powered insights.

## Overview

Upload a CSV or Excel file to instantly get:
- **Data Cleaning** - Automatic preprocessing and validation
- **Statistical Analysis** - Correlations, distributions, anomalies
- **Interactive Charts** - Real-time visualizations
- **AI Insights** - Natural language analysis powered by Groq API
- **Q&A Chat** - Ask questions about your data

Results generated in **5-10 seconds** with zero manual preprocessing.
